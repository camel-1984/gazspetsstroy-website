Репозиторий представляет собой описание реализованного коммерческого проекта для компании **ООО ГазСпецСтрой**.  
Проект выполнен в формате полного цикла: от проектирования интерфейса до развертывания в production.

## Объём работ

- разработка production-сайта для компании **ООО ГазСпецСтрой**;
- проектирование структуры и пользовательских сценариев;
- дизайн и верстка многостраничного интерфейса;
- адаптация под desktop, tablet и mobile;
- реализация интерактивных элементов и пользовательских сценариев;
- интеграция форм заявок с CRM Bitrix24;
- подключение Яндекс Метрики и API Яндекс Карт;
- SEO-подготовка: canonical, Open Graph, Schema.org / JSON-LD;
- оптимизация изображений и подготовка production-сборки;
- деплой проекта.
