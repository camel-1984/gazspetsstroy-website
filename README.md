## О проекте

Репозиторий представляет собой описание реализованного коммерческого проекта для компании **ООО ГазСпецСтрой**.  
Проект выполнен в формате полного цикла: от проектирования интерфейса до развертывания в production.

## Объём работ

- разработка дизайна и проектирование пользовательского интерфейса;
- верстка многостраничного сайта;
- реализация адаптивного интерфейса (desktop / tablet / mobile);
- добавление интерактивности с использованием JavaScript;
- SEO-подготовка (meta-теги, Open Graph, canonical);
- оптимизация медиа-ресурсов и внедрение формата WebP;
- сборка и подготовка проекта к production;
- настройка хостинга и деплой проекта.
